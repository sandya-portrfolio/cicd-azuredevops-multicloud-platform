# Multi-Cloud CI/CD Platform (Azure DevOps + AWS + Kubernetes)

## 🚀 Project Summary
Designed and implemented a multi-cloud CI/CD platform using Azure DevOps to automate application build, testing, containerization, and deployment across AWS and Azure environments. Leveraged Terraform for Infrastructure as Code (IaC) and Kubernetes for container orchestration. Supported hybrid workloads across Linux and Windows environments, ensuring high availability, scalability, and reliable production operations.

## 🔧 Key Responsibilities

### 🔹 Cloud Infrastructure & IaC
- Designed, provisioned, and maintained cloud infrastructure using Terraform across AWS and Azure environments  
- Managed cloud resources including compute, storage, networking, and IAM/security configurations  
- Ensured infrastructure consistency using modular Terraform architecture and environment-specific configurations  
- Supported Azure environments for basic infrastructure setup and troubleshooting  

---

### 🔹 CI/CD & Release Engineering (Azure DevOps)
- Designed and maintained CI/CD pipelines using Azure DevOps for automated build, test, and deployment workflows  
- Implemented multi-stage pipelines (Dev, QA, Prod) with approval gates and release controls  
- Automated Docker image build, tagging, and push to container registries (AWS ECR/Azure Container Registry)  
- Troubleshot pipeline failures, build issues, and deployment errors  
- Participated in release planning, validation, rollback, and deployment strategies  

---

### 🔹 Kubernetes & Containerization
- Deployed and managed containerized applications on Kubernetes (AWS EKS / Azure Kubernetes Service)  
- Managed Kubernetes resources including Deployments, Services, ConfigMaps, Secrets, and Namespaces  
- Handled application scaling, rolling updates, and zero-downtime deployments  
- Troubleshot issues such as pod failures, resource limits, and cluster-level problems  
- Collaborated with development teams to support Kubernetes-based application deployments  

---

### 🔹 Production Support & Incident Management
- Provided production support for cloud-hosted and containerized applications  
- Diagnosed and resolved issues across infrastructure, Kubernetes, CI/CD pipelines, and application layers  
- Participated in incident response, root cause analysis (RCA), and post-incident reviews  
- Maintained system uptime, reliability, and operational stability  

---

### 🔹 Operating Systems (Linux & Windows)
- Supported and managed Linux and Windows servers in cloud environments  
- Diagnosed system-level issues including service failures, performance degradation, and configuration errors  
- Worked closely with application teams to resolve OS-related incidents  

---

### 🔹 Monitoring & Observability
- Implemented monitoring and alerting using tools like CloudWatch and Azure Monitor  
- Created dashboards and alerts for system health, performance, and availability  
- Responded to alerts and proactively resolved potential issues  
- Identified recurring issues and implemented improvements to reduce operational risks

### 🔹 Collaboration & Continuous Improvement

- Maintained runbooks, documentation, and operational procedures

- Continuously improved automation, reliability, and deployment efficiency
